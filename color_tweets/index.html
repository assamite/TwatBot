Main package of the project, developed as a Django app, which has currently following modules.

Django App Modules:

  • models: Django models for resources given on the course.
  • views: Custom Django views for the app.
  • admin: Django admin functions.
  • tests: Basic tests for the app.

Modules explicit for TwatBot:

  • color_utils: Semantic agnostic color utility functions
  • resources_utils: Functions to populate Django models with contents in resources/
