Colorbot
================

Main package of the project, developed as a `Django <https://www.djangoproject.com/>`_ app.

Package has currently following modules:

* Basic Django app modules
	* :py:mod:`colorbot.models` - Django models for resources given on the course.
	* views

* Modules special for Colorbot
	* :py:mod:`colorbot.color_utils` - Utility functions for working with colors.
