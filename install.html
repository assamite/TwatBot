Installing¶

TwatBot is developed as a Django app. Access to Twitter API is gained using python-twitter.

You should be familiar with basic Django project layout and have a working Twitter app which allows you to tweet before using this project.

Short installing notes:

  • Install pip

  • Install third party libraries:

    $> cd project_root/
$> pip install -r requirements.txt // you might need root privileges

  • Create local settings files:

    $> cd project_root/TwatBot/
$> touch lsettings.py

  • Populate lsettings.py with at least following attributes (see Twitter API documentation for details):

    TWITTER_API_KEY = 'Your Twitter API key'
TWITTER_API_SECRET  = 'Your Twitter API secret'
TWITTER_ACCESS_TOKEN = 'Your Twitter access token'
TWITTER_ACCESS_TOKEN_SECRET = 'Your Twitter access token secret'
